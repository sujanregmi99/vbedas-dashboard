About This Project

Real-time vibration data is collected by two MPU6050 sensors connected to ESP32 microcontrollers. The nodes communicate via ESP-NOW. When both nodes detect vibration within a 5-second coincidence window, the event is confirmed — local alerts (buzzer, LED), SMS (SIM800L), and Firebase logging all activate.

Single-node detections are treated as local disturbances. Only dual-node coincidence events are escalated.

⚠ PROTOTYPE DEVICE — does not predict earthquakes or calculate Richter magnitude. Designed as a low-cost local awareness system for abnormal ground vibration.
